## Run Locally

Clone the project

```bash
  git clone https://github.com/shanibider/Web-Development-Projects.git
```

Go to the project directory

```bash
  cd "project-directory"
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm start / nodemon index.js
```
